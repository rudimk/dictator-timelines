Welcome to The Dictators' Timelines.

I've always been interested in using code to make the art of teaching history more...awesome. This project is a start on that - the idea is to compare the lifetimes of different dictators, using the TimelineJS library, from the fabulous folks at Northwestern University's Knight Lab.

Currently, this is a basic Github documentation site, with a timeline embedded in it. In a couple weeks, I intend to build a CMS on top of it, which allows you to add different timelines, dictators, and so on.
